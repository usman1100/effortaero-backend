<h1 align="center"><b>EffortAero</b> Backend</h1>


  <p align="center">
  A modern REST API architecture backend web service for the EffortAero Systems inc.
  </p>



<br/>


## Description

REST API server powered by NestJS and Express. 

<br/>


## Installation

```bash
$ yarn
```
OR
```bash
$ npm install
```

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```
